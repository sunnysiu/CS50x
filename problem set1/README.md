#### Implements a program that calculates the minimum number of coins required to give a user change.

```
$ ./cash0
[or]
$ ./cash1
Change owed: 0.41
4
```

### Usage

Program should behave per the example below.

NOTE: It is in USD curreny.

```
$ ./cash0
Change owed: 0.41
4
```

```
$ ./cash0
Change owed: -0.41
Change owed: -0.41
Change owed: foo

4
```
